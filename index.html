ChatGPT注册教程
2023/4/1

两个主要问题

第一，国内被墙的问题，中国、香港的不行，这里用的美国的，速度挺快

第二，国内手机号不能注册的问题，购买虚拟账号，大概一两块钱

chatgpt注册流程比较简单，根据提示进行即可

购买虚拟账号

据说有很多网站提供这项服务
使用<sms-activate.org>

  • 注册账号
  • 充钱
    我购买的时候，最低2美元，折合人民币大概14.5元，可用支付宝
  • 购买虚拟账号
    在搜索栏搜索openai，选择不同国家的账号，每个国家账号价格不同，我选择最便宜的印度尼西亚账号，大概人民币2元
  • 接收验证码
    购买后直接可以看到虚拟号码，将虚拟号码按照格式粘贴到openai上，稍后虚拟号码网站会收到验证码

    填入openai中，账号注册成功，可以开始和chatgpt对话啦

2023/4/4 登录chatgpt出现问题，access denied

openai禁了很多账号，据说集中在亚洲地区，参考这个博客解决了问题。节点从美国换成了日本，又可以对话了

hexo部署
2023/3/31

hexo + github部署

hexo安装

  • git安装
  • nodejs安装
  • hexo安装
    • 使用npm安装hexo npm install -g hexo-cli
    • 初始化博客 hexo init blog
    • 生成 hexo g= hexo generate
    • 本地预览 hexo s=hexo server
    • 打开localhost:4000预览

部署到github

  • github新建repo

    注意repo名为github用户名.github.io

    例如 用户名为hybai的用户 新建repo为hybai.github.io
  • 将hexo和github链接

    找到创建的博客文件夹下的_config.yml文件

    修改deploy
    deploy:
type: git
repo: git@github.com:c-haos-onfusion/c-haos-onfusion.github.io.git
branch: gh-pages
    repo用ssh的，不然会报错10054

    输入三条命令
    hexo clean 
hexo g 
hexo d
    hexo d=hexo deploy

绑定域名

  • 在阿里云买域名
  • 配置域名
  • github配置
  • 在source文件夹下创建CNAME文件，写入购买的域名
  • 输入三条命令
    hexo clean 
hexo g 
hexo d

github创建免费图床

因为写博客需要插入图片，需要一个稳定图床保存图片，获取url，采用github的repo做图床，搭配picgo
  • 下载picgo
  • github新建图床repo 随便命名
  • 配置picgo

    其中token在github中用户settings/developer setting里

    至此，图床创建好了
    第一天还好好的，第二天发现图片全没了，显示不出来，开始查原因
    右键单击检查，发现错误是net::ERR_NAME_NOT_RESOLVED

    网上说是因为github ip经常变，需要在host文件中手动添加ip，具体步骤如下：
  • 查找raw.githubusercontent.com的ip
    这里使用这个网站查ip

    把这几个ip挨个ping，找最快的
  • 修改host文件
    位置在C:\Windows\System32\drivers\etc\hosts

    在最后一行添加ip和raw.githubusercontent.com

    再刷新下网站，图片就加载出来了

    还可以用switchhosts管理ip

修改主题

这里采用particlex的主题
  • 可按照particlex主题的教程修改
  • 修改 theme: hexo-theme-particlex-master

    注意！只修改这里只能在本地预览到主题的变更，但deploy后网络上主题没变
    需要修改url `url: https://c-haos-onfusion.github.io/`
  • 主题修改的其他细节参见所使用主题的教程
